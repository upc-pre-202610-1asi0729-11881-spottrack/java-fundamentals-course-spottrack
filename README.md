# java-fundamentals-course-spottrack

Repositorio de código fuente del curso **"Fundamentos de Java para Principiantes"**, elaborado como parte del trabajo complementario del curso **1ASI0729 Desarrollo de Aplicaciones Open Source** (UPC, ciclo 202610).

🔗 **Repositorio**: [github.com/upc-pre-202610-1asi0729-11881-spottrack/java-fundamentals-course-spottrack](https://github.com/upc-pre-202610-1asi0729-11881-spottrack/java-fundamentals-course-spottrack)

Curso introductorio de programación con Java dirigido a estudiantes de secundaria (12 a 17 años) sin experiencia previa en código. Toda la experiencia de aprendizaje funciona **100% en el navegador**, sin necesidad de instalar nada.

📄 **Plan de curso completo**: [course-plan.md](./course-plan.md)

---

## Requisitos

Solo un navegador web (Chrome, Firefox, Safari o Edge). No se necesita instalar Java.

---

## Estructura del repositorio

```
.
├── README.md
├── course-plan.md
└── lessons/
    ├── lesson01-intro-java.md
    ├── lesson02-variables.md
    ├── lesson03-control-flow.md
    ├── lesson04-estructuras-datos.md
    ├── lesson05-metodos-io.md
    ├── lesson06-intro-poo.md
    ├── lesson07-fundamentos-poo.md
    ├── lesson08-programa-sencillo.md
    └── lesson09-consejos-finales.md
```

Cada archivo en `/lessons` corresponde a una lección del curso, e incluye en un solo `.md`:

- Descripción y conclusiones clave de la lección
- **Starter file**: código con espacios en blanco (`TODO`) para que el estudiante complete
- **Completed example**: versión final y funcional, para seguir como guía o usar como referencia

---

## Cómo usar este repositorio

1. Abre el archivo de la lección correspondiente en `/lessons`.
2. Copia el bloque de código del **starter file** en un editor en línea como [JDoodle](https://www.jdoodle.com/online-java-compiler).
3. Completa los `TODO` siguiendo el video de la lección.
4. Compara tu resultado con el bloque **completed example** del mismo archivo.
