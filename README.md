# java-fundamentals-course-[nombre-del-equipo]

Repositorio de código fuente del curso **"Fundamentos de Java para Principiantes"**, elaborado como parte del trabajo complementario del curso **1ASI0729 Desarrollo de Aplicaciones Open Source** (UPC, ciclo 202610).

Curso introductorio de programación con Java dirigido a estudiantes de secundaria (12 a 17 años) sin experiencia previa en código. Toda la experiencia de aprendizaje funciona **100% en el navegador**, sin necesidad de instalar nada.

📄 **Plan de curso completo**: [course-plan.md](./course-plan.md)

---

## Requisitos

Solo un navegador web (Chrome, Firefox, Safari o Edge). No se necesita instalar Java ni ningún editor.

## Estructura del repositorio

```
.
```

| Carpeta | Contenido |
|---|---|
| `/starter-files` | Archivos base con espacios en blanco (`TODO`) para que el estudiante los complete durante la lección. |
| `/completed-examples` | Versión final y funcional de cada ejercicio, para seguir como guía paso a paso o usar como referencia. |
| `/scripts` | Guiones en español de los videos, listos para grabación. |

---

## Cómo usar este repositorio

1. **Para practicar código Java** (variables, POO): copia el contenido de `/starter-files` en un editor en línea como [JDoodle](https://www.jdoodle.com/online-java-compiler), [Replit](https://replit.com) u [OnlineGDB](https://onlinegdb.com). Compara tu resultado con `/completed-examples` cuando termines.
2. **Para la lección de página web** (HTML/CSS): abre `completed-examples/index.html` y `completed-examples/style.css`, y pégalos en las columnas correspondientes de [CodePen](https://codepen.io).
3. **Si estás grabando una lección**: usa el guion correspondiente en `/scripts`.
